# PixiPlay - Mini Games Platform

PixiPlay is a web-based mini games platform built using **React**, **Spring Boot**, and **PostgreSQL**. It provides users with an engaging gaming experience through classic games, user authentication, score tracking, leaderboards, and a fun pixel-themed interface.
---

## Features

### User Authentication
- User Registration
- Secure Login
- User Profile with Pixel Avatar (DiceBear API)

### Mini Games
- Snake Game
- Sudoku
- Daily Trivia Challenge (Open Trivia Database API)

### Score Management
- Save game scores
- View current score

### User Interface
- Pixel-inspired theme
- Responsive design
- Cute pastel color palette
- Animated background decorations
